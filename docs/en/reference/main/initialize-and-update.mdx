# Initialize, update and reset

`init()` - The main instance method that initializes the calendar.

```js
const calendar = new VanillaCalendar(element, params);
calendar.init();
```

You can also update the calendar with the new settings at any time using the `update()` method.

```js
calendar.settings.lang = 'de-AT';
calendar.settings.iso8601 = false;
calendar.update();
```

Or reset using the `reset()` method.

```js
calendar.reset();
```
