# Темы

Календарь поддерживает темную и светлую тему оформления, одна из тем применяется по умолчанию в зависимости от настроек пользователя.

Если ваш сайт поддерживает только одну тему или вы хотите, чтобы ваш календарь выглядел по-своему, вы можете принудительно выбрать любую из тем.

Кстати, поскольку мы принудительно включаем одну тему, нам не нужно загружать лишний файл css.

Удаляем лишний импорт css и активируем темную тему.
