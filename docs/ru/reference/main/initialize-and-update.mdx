# Инициализировать, обновить и сбросить

`init()` - Основной метод экземпляра, который инициализирует календарь.

```js
const calendar = new VanillaCalendar(element, params);
calendar.init();
```

Вы также можете обновить календарь с новыми настройками в любое время, используя метод `update()`.

```js
calendar.settings.lang = 'de-AT';
calendar.settings.iso8601 = false;
calendar.update();
```

Или сбросить, используя метод `reset()`.

```js
calendar.reset();
```
